처음 설정 — Firebase 연결이 필요합니다 (1회만)
  1. Firebase 콘솔에서 프로젝트 추가 → 이름 예: fiti-quote (애널리틱스는 꺼도 됨)
  2. 왼쪽 메뉴 빌드 → Firestore Database → 데이터베이스 만들기 → 프로덕션 모드 → 위치 asia-northeast3 (서울)
  3. 빌드 → Authentication → 시작하기 → 로그인 방법에서 이메일/비밀번호 사용 설정 → Users 탭에서 팀 계정 추가 (예: 팀 공용 1개 또는 팀원별)
  4. 프로젝트 개요 옆 ⚙ 프로젝트 설정 → 일반 → 내 앱 → 웹 앱(</>) 추가 → 표시되는 firebaseConfig 값을 복사
  5. 이 HTML 파일을 메모장으로 열어 상단의 ★ FIREBASE CONFIG ★ 부분에 붙여넣기 → 저장 → 새로고침
  6. Firestore 규칙 탭에 아래 붙여넣고 게시 (로그인한 사람만 읽기/쓰기):
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
※ GitHub에 올릴 경우 저장소는 Private 권장. firebaseConfig 자체는 공개되어도 위 규칙 + 로그인이 있으면 데이터는 보호됩니다.
